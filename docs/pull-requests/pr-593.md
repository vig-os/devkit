## Summary

Consolidates four Renovate update branches that were stuck due to changelog-churn loops (empty `docs(changelog): add unreleased entry…` commits) and a `requires-python` patch bump the toolchain couldn't resolve. Each update is squash-merged into one clean commit:

- **#586** — `python:3.14-slim-bookworm` base image digest → `7e2f304…` (verified multi-arch index, keeps arm64 working per #578)
- **#587** — `taiki-e/install-action` digest → `bafb217`
- **#588** — GitHub Actions minor/patch (incl. `astral-sh/setup-uv` `0.11.19` → `0.11.21`)

Renovate bumped `requires-python` to `==3.14.6`, which the pinned `uv 0.11.19` could not download — blocking `uv sync` / pre-commit. The companion #588 bump (`setup-uv` → 0.11.21) fixes this in CI; merging the group together unblocks it. Verified locally by updating `uv` to 0.11.21 and installing the 3.14.6 interpreter.
