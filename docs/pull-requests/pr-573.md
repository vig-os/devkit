# [PR 573](https://github.com/vig-os/devcontainer/pull/573) chore(image): accept Debian won't-fix LOW CVEs in .trivyignore

## Description

Accept 78 unfixed LOW Debian OS-package CVEs from the next-release image into `.trivyignore` with a shared risk note and 2026-12-01 expiration. Adds a `check-expirations` utility and wires it into pre-commit, CI, and the nightly security scan so expired exception entries fail the pipeline (IEC 62304 exception register model). The GitHub Security tab LOW count will drop once `:latest` is refreshed to the post-buckets-B–D image.
