# [PR 1200](https://github.com/vig-os/devkit/pull/1200) fix(workspace): apply rsync excludes to --preview ADDED report

## Summary

`init-workspace.sh --preview` reported files as ADDED that the real copy then skipped (notably the legacy `.typos.toml` case from exo-pet/vault#31), because the preview classification didn't apply the mode/config-dependent excludes the rsync copy uses. The exclude decision is now a single SSoT array (`MODE_CONFIG_EXCLUDES`) consumed by **both** the `--preview` classifier and the rsync copy — so the preview is a faithful dry-run and the logic isn't duplicated (DRY).
