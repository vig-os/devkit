# [PR 570](https://github.com/vig-os/devcontainer/pull/570) chore: update vulnerable Python dependencies

## Description

Remediate Dependabot/Trivy advisories for vulnerable Python dependencies in this repo and the workspace template seeded to consumer repos (including smoke-test).

This repo bumps `urllib3`, `requests`, `idna`, and `Pygments` in `uv.lock`. The workspace template adds `[tool.uv] constraint-dependencies` floors for the jupyter stack (`notebook`, `jupyterlab`, `jupyter-server`, `mistune`) so downstream `uv.lock` files resolve patched versions on release re-sync.

`pytest` (GHSA-6w46-j5rx-g56g) is already at 9.0.3 via #528 — no action; its stale alert should auto-close.
